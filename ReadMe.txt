Al ejecutar el arhivo Compiler_G-2, se compila el programa que se encuentra en la carpeta Compilador/Testing/program.txt y genera el output en la carpeta Compilador/Output/out.txt.
Si se compila desde la consola de comandos de windows, se imprimiran todo el contenido final de la tablade simbolos al finalizar el programa.

Si se desea probar otro programa, se debe modificar el archivo program.txt ubicado en Compilador/Testing/program.txt
